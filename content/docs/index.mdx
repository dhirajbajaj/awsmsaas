awsmsaas is a solid starter boilerplate for your next saas app. 

It integrates Next.js 14, Prisma, Neon, Auth.js v5, Resend, React Email, Shadcn/ui, and Stripe all in one to get you started asap. 

This documentation is your go-to resource for configuring and using the starter effectively.

Let's get started and happy coding!
